# Car Rental System (Frontend)

## 📌 Description
This is a frontend-based Car Rental System project developed using HTML, CSS, and JavaScript. It allows users to view available cars and simulate the booking process. The main focus of this project is user interface design and basic functionality.

## 🚀 Features
- View available cars
- Car details display
- Booking form (UI based)
- Simple and user-friendly design

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
